Cómo ejecutar en un servidor local

Para ejecutar este sistema correctamente en Google Chrome, necesitas usar un servidor web local. Hay varias opciones:

Opción 1: Usando Python (más fácil)

  1. Guarda este archivo HTML como index.html
  2. Abre una terminal o línea de comandos
  3. Navega a la carpeta donde guardaste el archivo
  4. Ejecuta uno de estos comandos (según la versión de Python que tengas instalada):
    • Python 3: python -m http.server
    • Python 2: python -m SimpleHTTPServer
  5. Abre Chrome y ve a http://localhost:8000

Opción 2: Usando Visual Studio Code

  1. Instala la extensión "Live Server"
  2. Abre el archivo en VS Code
  3. Haz clic en "Go Live" en la barra de estado (esquina inferior derecha)

Opción 3: Usar aplicaciones de servidor local

  • XAMPP, WAMP, MAMP (según tu sistema operativo)
  • Coloca el archivo en la carpeta pública del servidor (generalmente "htdocs" o "www")
